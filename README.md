一个vscode插件，自用为主，使用ChatGPT进行增加自己开发速度和效率

## Extension Settings

- 安装插件后，打开设置，搜索UsefulCode，可以看到插件的设置项
- 或者windows下按下`ctrl+shift+p`，输入InputToken，设置OpenApiToken即可使用

## Known Issues
